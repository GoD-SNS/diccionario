# Data SLA (Acuerdo de Nivel de Datos).

Acuerdo formal entre proveedores y consumidores de datos que detalla las responsabilidades mutuas, respalda arquitecturas de datos confiables y evita ambigüedades entre los stakeholders. Define niveles de disponibilidad, exactitud y latencia de la actualización de los datos, asegurando que estos se entreguen en condiciones que habiliten decisiones confiables (Redman, 1996; Seiner, 2014; Loshin, 2010).

## Referencias

- Redman, T. (1996). Data Quality for the Information Age. Artech House.
- Seiner, R. S. (2014). Non-Invasive Data Governance. Technics Publications.
- Loshin, D. (2010). Master Data Management. Morgan Kaufmann.
