# Data Domain (Dominio de Datos).

Agrupación lógica de información que permite segmentar la gestión y gobernanza de datos, a partir de la definición de estructuras de propiedad, calidad y reglas de validación e integración (Marco, 2006; Loshin, 2013).

## Referencias

- Marco, D. (2006). Universal Meta Data Models. Wiley.
- Loshin, D. (2013). The Practitioner's Guide to Data Quality Improvement. Morgan Kaufmann.
