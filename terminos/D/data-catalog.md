# Data Catalog.

Es el motor de la alfabetización de datos, puesto que actúa como repositorio unificado, proporcionando contexto, linaje, calidad y gobernanza, lo que facilita el descubrimiento, comprensión y uso de los datos (Gartner, 2020; Eckerson, 2018; Loshin, 2023).

## Referencias

- Gartner. (2020). Data Virtualization: A Modern Data Integration Approach.
- Eckerson, W. (2018). The Rise of Logical Data Warehouses and Data Virtualization. Eckerson Group.
- Loshin, D. (2013). The Data Management Toolkit. Morgan Kaufmann.
