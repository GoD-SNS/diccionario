# Data Strategy.

Marco conceptual, técnico y organizativo, cuya implementación se da a través de políticas de calidad, lineamientos de acceso, estructuras de roles y el uso de plataformas que aseguren el uso de los datos como un activo estratégico para el desarrollo de los objetivos institucionales (Marr, 2017; Ladley, 2019).

## Referencias

- Ladley, J. (2019). Data Governance: How to Design, Deploy and Sustain an Effective Data Governance Program (2nd ed.). Academic Press.
