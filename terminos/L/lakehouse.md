# Lakehouse

Arquitectura híbrida que combina las bondades de los data lakes y data warehouses, permitiendo a las empresas consolidar una infraestructura de datos robusta, análoga al procesamiento estructurado de estos. Lo que permite la integración de gobernanza y calidad bajo una flexibilidad analítica con menor complejidad técnica (Armbrust, 2021; Eckerson Group, 2021)
