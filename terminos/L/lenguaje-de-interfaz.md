# Lenguaje de Interfaz.

Se refiere a los mecanismos que permiten al usuario comunicarse con el sistema de visualización de datos. Estas interfaces interpretan consultas expresadas por usuarios y las transforman en instrucciones ejecutables, lo que mejora el uso y la accesibilidad de los sistemas BI facilitando la interacción sin necesidad de conocimientos técnicos (Zhang et al., 2024).
