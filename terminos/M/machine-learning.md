# Machine Learning.

Disciplina que se centra en el desarrollo de métodos y algoritmos capaces de aprender de datos observados, generando modelos predictivos sin programación explícita, mediante la optimización de funciones objetivo. Su estructura común involucra la representación del problema, la elección de un criterio de aprendizaje y la evaluación del modelo para asegurar su capacidad de generalización (Jung, 2022).

## Referencias

- Jung, A. (2022). Machine Learning: Basic Principles (arXiv:1805.05052v6). arXiv.
