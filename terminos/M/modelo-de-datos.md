# Modelo de Datos.

Representación abstracta que sirve como columna vertebral de cualquier sistema analítico, facilitando el desarrollo de soluciones robustas, escalables y mantenibles a largo plazo. Orquesta la organización y estructura de entidades, atributos y relaciones entre los datos de un sistema, lo que le convierte en la base para la interoperabilidad entre unidades del negocio (Silberschatz, Korth & Sudarshan, 2020; DAMA International, 2017)
