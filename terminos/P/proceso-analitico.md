# Proceso Analítico.

El proceso analítico es comprendido como una variante específica del método científico, dedicada al examen de las múltiples formas del discurso. Dicho proceso exige disposiciones de atención crítica, interpretación reflexiva y capacidad de acción (Lopera, Ramírez, Zuluaga & Ortiz, 2010). Es decir, partir de un problema general se analiza para llegar a las causas, llegando a las partes elementales para entender relaciones causales y así obtener resultados precisos.
