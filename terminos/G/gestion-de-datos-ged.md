# Gestión de datos (GeD)

Sistema que almacena y procesa información de manera planificada y constante que permite ordenar de manera descriptiva los datos que se producen en una investigación. Al seguir políticas y buenas prácticas, se asegura que estos datos no solo estén disponibles cuando se necesitan, sino que también puedan ser entendidos y utilizados por otros en el futuro, promoviendo así la transparencia, la colaboración y el aprovechamiento responsable del conocimiento (Campos Jara, 2023). Se comprende como el proceso de desarrollo, ejecución y supervisión de planes, políticas, programas y prácticas que controlan, protegen, entregan y mejoran el valor de los datos y activos de información. (DMBOK).
