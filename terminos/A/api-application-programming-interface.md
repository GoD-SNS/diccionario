# API (Application Programming Interface).

Activo digital estratégico que define cómo los sistemas de software interactúan mediante un conjunto de especificaciones técnicas, lo que permite construir ecosistemas tecnológicos interconectados. Esto facilita la integración de soluciones analíticas con fuentes internas y externas, estandarizando el acceso a datos, orquestando flujos automatizados, implementando microservicios analíticos y simplificando la interoperabilidad entre plataformas (Jacobson, Brail & Woods, 2012; Villamizar, Garcés & Casallas, 2015; Microsoft Azure, 2021).

## Referencias

- Jacobson, D., Brail, G., & Woods, D. (2012). APIs: A Strategy Guide. O'Reilly Media.
- Villamizar, M., Garcés, O., & Casallas, R. (2015). Evaluating the Performance of Web APIs. IEEE Latin America Transactions.
- Microsoft Azure. (2021). Designing RESTful APIs for Data Services.
