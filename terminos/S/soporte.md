# Soporte

Asistencia técnica y funcional que se proporciona a usuarios y sistemas para garantizar el correcto funcionamiento de las plataformas de datos, resolver incidencias, responder consultas y facilitar el uso efectivo de los activos de información. En gobernanza de datos, incluye el apoyo para el cumplimiento de políticas y el uso adecuado de herramientas analíticas.
