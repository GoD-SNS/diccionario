# Tablero

Un tablero (o dashboard) es una interfaz interactiva diseñada para visualizar indicadores clave y métricas mediante visualizaciones graficas, con el fin de facilitar una toma de decisiones con criterio. Además, sirve como el principal punto de contacto entre el usuario y el sistema de BI, donde una distribución adecuada del contenido reduce la carga cognitiva, facilitando la integración efectiva por parte de quienes toman las decisiones (Ansorge et al., 2024).
