# Tabla Dinámica.

Herramienta interactiva de resumen de datos, cuya función es clave para crear prototipos rápidos de análisis, a través de funciones que permiten reorganizar, agrupar, filtrar y calcular estadísticas, sin necesidad de programación avanzada (Walkenbach, 2013; Kusleika, 2021).

## Referencias

- Walkenbach, J. (2013). Excel 2013 Power Programming with VBA. Wiley.
- Kusleika, D. (2021). Microsoft Excel 365 Formulas and Functions. Microsoft Press.
