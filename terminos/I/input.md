# Input

El input (o datos de entrada) comprende el conjunto de datos que se integran desde múltiples fuentes (como sistemas financieros, APIs, archivos) y que forman la base del proceso ETL. La eficacia de esta etapa es crucial para lograr reportes y análisis confiables en tiempo real, ya que permite reducir la latencia y mejorar la calidad de los datos gracias a métodos adaptativos de integración en flujo continuo (Saha et al., 2019).
