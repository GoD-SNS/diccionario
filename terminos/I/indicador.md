# Indicador

Un indicador (o KPI) es una métrica que cuantifica para evaluar el desempeño de procesos y metas específicas de una organización. Estos indicadores se construyen sobre uno o varios datos básicos que permiten monitorear si se están cumpliendo los objetivos, alineando esta medición con el contexto del proyecto para facilitar una toma de decisiones y la planificación (Novak et al., 2012).
