# Ingeniero de Datos.

Rol encargado de garantizar la infraestructura necesaria para construir y mantener sistemas que permitan la gestión de los flujos de datos, estos flujos permiten la recolección, almacenamiento, procesamiento y entrega eficiente de grandes volúmenes de datos para su posterior análisis, utilidad y gobernanza (Gutierrez, 2022).
