# Big Data.

Conjunto de datos el cual tiene un tamaño, velocidad de generación y variedad tan grandes que sobrepasa la capacidad de procesamiento de los sistemas convencionales. Para el manejo de esta data se requieren tecnologías cualificados para capturar, almacenar, analizar y visualizar datos estructurados, no estructurados o semiestructurados. Su valor radica en la capacidad de extraer información útil para el apoyo en la toma de decisiones (Alwan y Ku-Mahamud, 2020).
