# Bodegas de datos (Data WareHousing).

Arquitectura de información o colección de datos diseñada para consolidar información de múltiples fuentes. Su propósito se basa en proveer datos confiables para su posterior análisis, logrando un respaldo en el proceso de la toma de decisiones (Inmon, 2005; Oracle Corporation 2020). Son sistemas que almacenan grandes cantidades de datos históricos de diferentes fuentes, organizados para facilitar el análisis y la generación de informes.

## Referencias

- Inmon, W. H. (2005). Building the Data Warehouse (4th ed.). Wiley.
- Oracle Corporation. (2020). Oracle Data Warehousing Guide.
