# Estadística.

Es una disciplina que permite extraer información de una población a partir de muestras, aplicando métodos rigurosos para recopilar, interpretar y analizar los datos. Esto con el fin de estimar patrones y tendencias para llegar a inferencias con alta confiabilidad (UNAM, s. f.).

## Referencias
